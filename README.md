# Galactic Attack
A simple Space Invader-esque type of game created using c++ and glut.

# Controls
W - moves up <br/>
A - moves left <br/>
S - moves down <br/>
D - moves right <br/>
F - toggle fullscreen <br/>
P - pause game <br/>
R - resumes game <br/>
Z - restart <br/>
ESC - quit <br/>
